# Retail Sales & Profit Analytics System

## 1. Project Introduction

This project analyzes retail sales data using Python, SQL, and Tableau to identify business insights related to sales, profits, customer behavior, product performance, and shipping efficiency.

The project uses the Superstore retail dataset and demonstrates complete data analytics workflow including:

- Data Cleaning
- Exploratory Data Analysis
- Business Intelligence
- SQL Analytics
- Data Visualization
- Dashboard Development

---

## 2. Objectives

The main objectives of this project are:

- Analyze sales and profit performance
- Identify high-performing categories and regions
- Detect loss-making products
- Understand customer purchasing behavior
- Study shipping efficiency
- Build an interactive Tableau dashboard

## 12. Business Insights

- Technology products drive most revenue.
- Western region performs best overall.
- Certain furniture products reduce profitability.
- Customer concentration impacts total revenue.
- Shipping performance affects customer experience.

---

## 13. Recommendations

- Focus on profitable categories.
- Reduce losses from low-margin products.
- Improve shipping efficiency.
- Target high-value customers.
- Monitor regional performance regularly.
