# 🛒 Retail Sales Analytics Project

## 📌 Project Overview

This project analyzes retail sales data using:

- Python
- Pandas
- PostgreSQL
- SQL
- Matplotlib

The goal of the project is to extract business insights from retail sales data and build a professional end-to-end analytics workflow.

---

# 🚀 Technologies Used

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- Matplotlib
- Jupyter Notebook
- Git & GitHub
